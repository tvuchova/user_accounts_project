Welcome to this demo application!It's a small web application that manages user accounts in a database.
The application has the following functionality:

  • A list page that shows all user accounts in a sortable grid.
  • You can create new account dialog that adds new accounts to the database.
  • The following properties are supported: First name,Last name,Email address,Date of birth
  • A Delete account dialog that removes accounts from the database.
  • An Edit account dialog that allows to change account data of existing accounts.
